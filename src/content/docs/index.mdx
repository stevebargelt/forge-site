## Welcome to Forge

Forge orchestrates teams of AI agents to complete complex engineering work — reliably, transparently, and at a pace no single model can match.

### Core concepts

| Concept | Description |
|---------|-------------|
| **Runs** | Top-level units of work — a feature, fix, or investigation |
| **Tasks** | Individual steps within a run, each owned by a specialist agent |
| **Gates** | Checkpoints that require review (human or automated) before work advances |
| **Reds** | Adversarial reviewer agents that look for problems in completed work |
| **Agents** | Specialist containers: architect, engineer, test-engineer, red, and more |
| **Blackboard** | The SQLite database that records every decision and output |
